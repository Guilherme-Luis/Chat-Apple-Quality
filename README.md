# Chat Apple Quality

## Sobre o Projeto
Este projeto visa desenvolver um modelo de aprendizado de máquina para avaliar a qualidade de maçãs através de características específicas. O sistema utiliza um conjunto de dados para prever e classificar a qualidade das maçãs.

## Tecnologias Utilizadas
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Estrutura do Projeto
```
Chat-Apple-Quality/
├── data/                # Arquivos de dados
├── notebooks/           # Jupyter notebooks
├── src/                 # Código fonte
└── README.md           # Este arquivo
```

## Como Executar
1. Clone o repositório
```bash
git clone https://github.com/Guilherme-Luis/Chat-Apple-Quality.git
```

2. Instale as dependências
```bash
npm install
```

3. Execute o aplicativo:
```bash
npm start
```
## Dataset
O conjunto de dados contém informações sobre características de maçãs, incluindo:
- Tamanho
- Peso
- Doçura
- Crocância
- E outras métricas relevantes
